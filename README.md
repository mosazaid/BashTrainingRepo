# BashTrainingRepo

Comprehensive Shell & Bash scripting training for developers — especially Android (Gradle) and Java Spring Boot workflows on **macOS** and **Windows**.

## Documentation

| Guide | Description |
|-------|-------------|
| [Shell & Bash Scripting: Zero to Advanced](docs/shell-and-bash-scripting-guide.md) | Full reference from first terminal command through advanced scripting, CI/CD, Android `adb`/Gradle, and Spring Boot/Maven automation |

## What's Covered

- Terminal setup: macOS (zsh/bash), Windows (Git Bash, WSL2)
- Core shell: navigation, pipes, text processing, variables, control flow
- Production scripts: `set -euo pipefail`, traps, error handling, ShellCheck
- Advanced: arrays, parameter expansion, process substitution, sed/awk/xargs
- Platform portability across Mac, Linux, and Windows
- Android developer workflows (`adb`, emulator, Gradle wrapper)
- Spring Boot / Java workflows (SDKMAN, Maven wrapper, Docker, actuator health checks)
- CI/CD patterns for GitHub Actions and Jenkins

## Quick Start

```bash
# Read the guide
less docs/shell-and-bash-scripting-guide.md

# Or open in your editor
code docs/shell-and-bash-scripting-guide.md
```
